# STM32 Person 2 Receiver Project

## Overview
This project implements the receiver side of an embedded communication system using STM32 and UART/RS485.

## Features
- Displays "=== PERSON 2 READY ==="
- Receives "Hello"
- Transmits "how are you?"
- Blinks external LED

## Hardware
- STM32 Nucleo-L432KC
- Breadboard
- RS485 module
- LED and resistor

## Output
=== PERSON 2 READY ===  
Received: Hello  
Transmitted: how are you?
